            Overview
            Hand Tracking Mouse is an innovative input device that uses computer vision to detect hand position and gestures, converting them into mouse cursor movements and clicks. This hands-free interface provides an alternative way to interact with computers, particularly useful for presentations, accessibility, or gaming.
